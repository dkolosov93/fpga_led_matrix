Script for converting a PNG image to .mif format for Altera FPGA devices

Requires a .png image in the folder named: "input"
Creates output .mif file named: "image"

Default resolution is 32 x 32 (=1024)
WidtH: 24 bits per pixel ( RRRRRRRRGGGGGGGBBBBBBB )

To edit resolution edit file and change: W/H/WIDTH accordingly.
